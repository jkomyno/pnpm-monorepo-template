# `@jkomyno/common-utils`

Example publishable library from [`pnpm-monorepo-template`](https://github.com/jkomyno/pnpm-monorepo-template).
It demonstrates dual ESM and CommonJS output, declaration files, package exports, and source-first workspace resolution.

```ts
import { sum } from '@jkomyno/common-utils'

sum([1, 2, 3]) // 6
```
